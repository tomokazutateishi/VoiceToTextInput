# 会話メモ 2026-02-27

## トピック
VoiceToTextInput の UI デザイン調整

## 会話の流れ
- メニューバーアイコンが表示されない問題 → メニュー機能を削除し、Dock と ⌥ Space で操作
- 半透明ブラー・透過の要望 → NSVisualEffectView で実装を試みたが、期待通りに反映されないケースあり
- テキストエリアは変えず、ごく薄い色味のみに変更
- 録音停止時のインジケーターを青に変更（背景との区別を明確に）

## 気づき・学び
- macOS のメニューバーアイコンは Control Center で隠れることがある
- NSVisualEffectView の効果は環境やウィンドウ構成によって見え方が変わる
- ユーザー要望の「テキストエリア以外の透過」と「テキストエリアは変えない」の両立は、レイアウトの分離が必要

## NOTE ネタ候補

- macOS アプリの半透明 UI 実装の難しさ
