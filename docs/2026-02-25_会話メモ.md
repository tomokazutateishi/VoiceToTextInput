# 会話メモ 2026-02-25

## トピック
VoiceToTextInput（音声→テキスト入力）macOS アプリの開発

## 会話の流れ
- Notta・SuperWhisper のようなバイブコーディング向けアプリの作成依頼
- 複数役割で相談しながら安全・効率的に完成まで進める方針
- 実装: Swift + Speech Framework + クリップボード貼り付け

## 気づき・学び
- macOS 15 Sequoia では CGEvent にタイムスタンプ設定が必要（`clock_gettime_nsec_np`）
- NSPanel の `.nonactivatingPanel` でフォーカスを奪わずにウィンドウ表示可能
- 貼り付けはアプリによって挙動が異なり、Accessibility API が効かないケース（Cursor、VS Code 等）ではクリップボード + Cmd+V が基本
- アクセシビリティ権限のプロンプトは、一度許可済みでも毎回出る場合があり、ユーザー体験に影響
